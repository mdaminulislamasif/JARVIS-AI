───────────────────────────────────────────────────────────────────────────────

  - pip install -e .

Required Files:

  - pyproject.toml

  1. Create setup.py or pyproject.toml

  3. Run build command
  4. Install package

📊 EXAMPLES / উদাহরণ
═══════════════════════════════════════════════════════════════════════════════

EXAMPLE 1: Complete Development Cycle
───────────────────────────────────────────────────────────────────────────────
```python
from jarvis_development_master import DevelopmentMaster

dev = DevelopmentMaster()

project_path = '/path/to/my/project'
language = 'python'

# 1. Learn skills
print("Step 1: Learning...")
dev.learn_to_build(language)
dev.learn_to_develop('all')
dev.learn_to_test(language)
dev.learn_to_operate('all')

# 2. Build project
print("\nStep 2: Building...")
build_result = dev.build_project(project_path, language)
if build_result['status'] == 'success':
    print("✅ Build successful!")
else:
    print("❌ Build failed!")
    exit(1)

# 3. Test project
print("\nStep 3: Testing...")
test_result = dev.test_project(project_path, language)
if test_result['tests_failed'] == 0:
    print("✅ All tests passed!")
else:
    print(f"⚠️ {test_result['tests_failed']} tests failed")

# 4. Deploy/Operate
print("\nStep 4: Deploying...")
deploy_result = dev.operate_project(project_path, 'start')
if deploy_result['status'] == 'success':
    print("✅ Deployment successful!")

# 5. Monitor
print("\nStep 5: Monitoring...")
monitor_result = dev.operate_project(project_path, 'monitor')
print(monitor_result)

# Show statistics
print("\nStatistics:")
stats = dev.get_statistics()
for key, value in stats.items():
    print(f"  {key}: {value}")
```
