│                         TROUBLESHOOTING                            │
└────────────────────────────────────────────────────────────────────┘


    Problem: Personal data found
    ┌─────────────────────────────────────────────────────────────┐
    │  Solution:                                                  │
    │  1. Review the report                                       │
    │  2. Remove or replace sensitive data                        │
    │  3. Add files to .gitignore                                 │
    │  4. Run check again                                         │
    └─────────────────────────────────────────────────────────────┘

    Problem: Wrong repository URL
    ┌─────────────────────────────────────────────────────────────┐
    │  Solution:                                                  │
    │  1. Check URL format                                        │
    │  2. Must end with .git                                      │
    │  3. Copy from GitHub page                                   │
    │  4. Try again                                               │
    └─────────────────────────────────────────────────────────────┘

    Problem: Upload failed
    ┌─────────────────────────────────────────────────────────────┐
    │  Solution:                                                  │
    │  1. Check internet connection                               │
    │  2. Verify GitHub account                                   │
    │  3. Check authentication                                    │
    │  4. Try again                                               │
