💡 TIPS / টিপস
═══════════════════════════════════════════════════════════════════════════════

TIP 1: Use MRX button for ALL systems at once
       সব systems একসাথে চালাতে MRX button ব্যবহার করুন

TIP 2: Use commands for specific systems
       নির্দিষ্ট system এর জন্য commands ব্যবহার করুন

TIP 3: Don't type just numbers (1, 2, 3)
       শুধু numbers (1, 2, 3) টাইপ করবেন না

TIP 4: Type full command or use MRX button
       পুরো command টাইপ করুন অথবা MRX button ব্যবহার করুন

TIP 5: MRX button is RED and easy to find
       MRX button লাল রঙের এবং সহজে খুঁজে পাওয়া যায়
