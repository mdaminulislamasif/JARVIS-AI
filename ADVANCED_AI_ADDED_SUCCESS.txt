8 Types of Memory:

1. Short-Term Memory
   - Duration: 1-2 hours
   - Stores: Current conversation, active tasks
   - Like: Human working memory

2. Long-Term Memory
   - Duration: Permanent
   - Stores: User profile, preferences, history
   - Like: Human long-term memory

3. Episodic Memory
   - Stores: Specific events with time/context
   - Example: "You asked about Python on May 1st"
   - Like: Autobiographical memory

4. Semantic Memory
   - Stores: General knowledge and facts
   - Example: "You prefer dark mode"
   - Like: Factual knowledge

5. Procedural Memory
   - Stores: Skills and routines
   - Example: "You check email at 9 AM"
   - Like: Muscle memory

6. Working Memory
   - Stores: Active processing
   - Example: Calculations, comparisons
   - Like: Mental workspace

7. Memory Consolidation
   - Process: Transfer short-term → long-term
   - Automatic: Important info saved
   - Like: Sleep consolidation

8. Memory Retrieval
   - Process: Access stored memories
   - Fast: < 1 second
   - Like: Recall and recognition
