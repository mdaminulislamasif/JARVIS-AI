## 🔧 কিভাবে কাজ করে (HOW IT WORKS)

### ১. All Functions Panel বাটন ক্লিক ফ্লো:

```
ইউজার বাটন ক্লিক করে
    ↓
বাটন কল করে: lambda cmd=command: self._execute_command(cmd)
    ↓
_execute_command() thread তৈরি করে
    ↓
Thread কল করে: self.process_callback(command)
    ↓
process_callback = self.process (jarvis_panel.py থেকে)
    ↓
process() method command হ্যান্ডেল করে
    ↓
Command execute হয় এবং result দেখায়
```

### ২. Auto Learning বাটন ফ্লো:

```
ইউজার "AUTO BG LEARN" বাটন ক্লিক করে
    ↓
বাটন command পাঠায়: "autobg"
    ↓
process() method "autobg" receive করে
    ↓
Check করে যদি cmd_root == "autobg"
    ↓
যদি auto_bg_learner.is_running:
    → auto_bg_learner.stop() কল করে
    → "Auto Learning STOPPED" message দেখায়
নাহলে:
    → auto_bg_learner.start() কল করে
    → "Auto Learning STARTED" message দেখায়
    → Background learning thread শুরু করে
```

### ৩. Neural Protocols বাটন ফ্লো:

```
PASTE বাটন:
    → paste_key() কল করে
    → Clipboard থেকে API key paste করে
    → Masked key দেখায় (AIzaSy****ubfY)

SYNC বাটন:
    → sync_key() কল করে
    → API key validate এবং save করে
    → নতুন key দিয়ে brain reinitialize করে
    → Masked key দেখায়

PING বাটন:
    → ping_key() কল করে
    → Gemini দিয়ে API key test করে
    → PING OK বা PING FAIL দেখায়

TEST বাটন:
    → test_keys() কল করে
    → Diagnostic popup খোলে
    → Pool এর সব key test করে
    → প্রতিটি key এর status দেখায়

MODEL বাটন:
    → rotate_brain_model() কল করে
    → Model গুলোর মধ্যে switch করে
    → Model info display update করে

RESET বাটন:
    → async_init_brain() কল করে
    → Brain reinitialize করে
    → সব API key reload করে
```

           if self.process_callback:
               print(f"🔍 DEBUG: process_callback কল করছি {command} দিয়ে")
