TECHNICAL DETAILS / প্রযুক্তিগত বিবরণ:
---------------------------------------

File: jarvis_ultimate_learner.py

Classes:
- UltimateLearner - Main learning system

Methods:
- learn_everything(topic) - Learn everything about topic
- learn_multiple_topics(topics_list) - Batch learning
- get_all_knowledge() - List all knowledge
- get_statistics() - Show statistics

Database Tables:
- ultimate_knowledge - Stores all learned content
  - topic, category, content, source_url
  - word_count, importance, tags, learned_date
- learning_sessions - Tracks learning sessions
  - topics_learned, total_words, duration

Learning Sources:
- Wikipedia API
- Google Search
- Stack Overflow
- Web scraping

Chrome Integration:
- Finds Chrome on system
- Opens Chrome with search
- Uses os.startfile() for Windows
