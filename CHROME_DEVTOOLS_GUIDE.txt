EXAMPLE 4: Extract Search Results
----------------------------------
```javascript
let results = [];
document.querySelectorAll('div.g').forEach(function(result) {
    let title = result.querySelector('h3');
    let snippet = result.querySelector('.VwiC3b');
    if (title && snippet) {
        results.push({
            title: title.innerText,
            snippet: snippet.innerText
        });
    }
});
results;
```

TROUBLESHOOTING / সমস্যা সমাধান:
---------------------------------

Problem 1: ChromeDriver not found
Solution: Download ChromeDriver and add to PATH

Problem 2: Chrome version mismatch
Solution: Update ChromeDriver to match Chrome version

Problem 3: DevTools not opening
Solution: Check if pyautogui is installed correctly

Problem 4: JavaScript not executing
Solution: Wait for page to load completely

Problem 5: Selenium errors
Solution: Update selenium package:
