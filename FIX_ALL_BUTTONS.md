## 🐛 IF BUTTON NOT WORKING | যদি বাটন কাজ না করে

### Check 1: Is Panel Running?
```bash
python jarvis_panel.py
```

### Check 2: Is Button Clicked?
- Click button
- Look at terminal for output
- Should see: `[ROOT]> command_name`

### Check 3: Check Dependencies
Some buttons need specific modules:
- **Generator buttons**: Need AI models
- **Network buttons**: Need admin rights
- **Learning buttons**: Need internet

### Check 4: Check Logs
Look in terminal for error messages:
- `[ERROR]>` = Error occurred
- `[SYSTEM]>` = System message
- `[JARVIS]>` = Response

**Solution**:
- Button might be disabled
- Check if feature is available
- Restart JARVIS

- Check system resources
