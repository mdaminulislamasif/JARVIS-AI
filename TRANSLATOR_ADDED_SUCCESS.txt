================================================================================
  🚀 HOW TO USE | কিভাবে ব্যবহার করবেন
================================================================================

METHOD 1: Using Translation Dialog (Easiest)
পদ্ধতি ১: অনুবাদ ডায়ালগ ব্যবহার করে (সবচেয়ে সহজ)
----------------------------------------

1. Run JARVIS Panel:
   জার্ভিস প্যানেল চালান:
   python jarvis_panel.py

2. Click "🌍 TRANSLATE" button in left sidebar
   বাম সাইডবারে "🌍 TRANSLATE" বাটনে ক্লিক করুন

3. Translation dialog opens:
   অনুবাদ ডায়ালগ খোলে:
   • Enter text to translate
     অনুবাদ করার জন্য টেক্সট লিখুন
   • Select target language (en, bn, hi, etc.)
     লক্ষ্য ভাষা নির্বাচন করুন (en, bn, hi, ইত্যাদি)
   • Click "🌍 TRANSLATE" button
     "🌍 TRANSLATE" বাটনে ক্লিক করুন

4. See translation instantly!
   তাৎক্ষণিক অনুবাদ দেখুন!


METHOD 2: Using Command Line
পদ্ধতি ২: কমান্ড লাইন ব্যবহার করে
----------------------------------------

Translate text:
টেক্সট অনুবাদ করুন:

python jarvis_translator.py "text" target_lang [source_lang]

Examples:
উদাহরণ:

# English to Bengali
python jarvis_translator.py "Hello World" bn

# Bengali to English
python jarvis_translator.py "আমি জার্ভিস" en

# French to English
python jarvis_translator.py "Bonjour" en fr

# Spanish to Bengali
python jarvis_translator.py "Hola amigo" bn es


View supported languages:
সমর্থিত ভাষা দেখুন:

python jarvis_translator.py languages


View translation history:
অনুবাদ ইতিহাস দেখুন:

python jarvis_translator.py history


METHOD 3: Using Chat Commands
পদ্ধতি ৩: চ্যাট কমান্ড ব্যবহার করে
----------------------------------------

Type in JARVIS chat:
জার্ভিস চ্যাটে টাইপ করুন:

• translate      - Opens translation dialog
  translate      - অনুবাদ ডায়ালগ খোলে

• languages      - Shows supported languages
  languages      - সমর্থিত ভাষা দেখায়

• transhistory   - Shows translation history
  transhistory   - অনুবাদ ইতিহাস দেখায়
