⚙️ HOW IT WORKS / কিভাবে কাজ করে
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Random Topic Selection
  - Randomly selects a category
  - Randomly selects a topic from that category
  - Checks if already learned recently (last 7 days)

STEP 2: Learning with All 4 Systems
  - System 1: Internet Learner (Wikipedia + Web)
  - System 2: Ultimate Learner (Chrome + Google)
  - System 3: Auto Learner (Word by Word)
  - System 4: Chrome DevTools (Advanced)

STEP 3: Save to Database
  - Saves topic, category, timestamp
  - Saves which systems were used
  - Marks as success or failure

STEP 4: Wait and Repeat
  - Waits 5-10 minutes (random)
  - Selects next topic
  - Repeats forever until stopped

═══════════════════════════════════════════════════════════════════════════════
⏱️ TIMING / সময়
═══════════════════════════════════════════════════════════════════════════════

PER TOPIC:
  - Internet Learner: 3 seconds
  - Ultimate Learner: 3 seconds
  - Auto Learner: 3 seconds
  - Chrome DevTools: 3 seconds
  - Total: ~12 seconds per topic

BETWEEN TOPICS:
  - Wait time: 5-10 minutes (random)
  - This prevents overloading the system

PER HOUR:
  - Approximately 5-6 topics learned
  - 60-72 topics per 12 hours
  - 120-144 topics per 24 hours

💾 DATA STORAGE / ডেটা সংরক্ষণ
═══════════════════════════════════════════════════════════════════════════════

DATABASE TABLE: auto_background_learned
  - id: Unique ID
  - topic: Topic name
  - category: Category name
  - learned_at: Timestamp
  - systems_used: Which systems were used
  - success: 1 if successful, 0 if failed

📊 STATISTICS / পরিসংখ্যান
═══════════════════════════════════════════════════════════════════════════════

To see statistics, you can query the database:

TOTAL TOPICS LEARNED:
  SELECT COUNT(*) FROM auto_background_learned WHERE success = 1

TOPICS BY CATEGORY:
  SELECT category, COUNT(*) as count
  FROM auto_background_learned
  WHERE success = 1
  GROUP BY category
  ORDER BY count DESC

RECENT TOPICS:
  SELECT topic, category, learned_at, systems_used
  FROM auto_background_learned
  WHERE success = 1
  ORDER BY learned_at DESC
  LIMIT 10
