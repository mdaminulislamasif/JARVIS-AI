## 🎯 ব্যবহার (খুবই সহজ!)

### পদ্ধতি:

#### 1. Button Click করুন
JARVIS panel এ **"⚡ WORLD AI CHAT (AUTO)"** button খুঁজুন এবং click করুন।

#### 2. AI Select করুন
একটি dialog খুলবে। যেকোনো AI select করুন:
- 🔷 Google Gemini ⚡
- 🤖 ChatGPT ⚡
- 🧠 Claude AI ⚡
- 💬 Microsoft Copilot ⚡
- 🔮 Perplexity AI ⚡
- এবং আরও ৫টি!

#### 3. Browser খুলবে
- Browser automatically খুলবে
- AI website load হবে
- JARVIS panel এ message দেখাবে: "⚡ Type your question and press Enter!"

#### 4. JARVIS Panel এ লিখুন
- JARVIS এর chat box এ আপনার প্রশ্ন লিখুন
- যেমন: "What is Python programming?"
- অথবা বাংলায়: "Python programming কি?"

#### 5. Enter চাপুন
- Enter key চাপুন
- JARVIS automatically:
  - ✅ Browser এ প্রশ্ন পাঠাবে
  - ✅ AI response এর জন্য wait করবে
  - ✅ Response নিয়ে আসবে
  - ✅ JARVIS panel এ দেখাবে
  - ✅ Response speak করবে

#### 6. সম্পন্ন! 🎉
- Response JARVIS panel এ দেখা যাবে
- JARVIS response speak করবে
- আপনি আরও প্রশ্ন করতে পারবেন!


**কারণ**: Response এর জন্য wait করছে

**সমাধান**:
- কিছুক্ষণ wait করুন (10-20 seconds)
- Browser window check করুন
- যদি stuck থাকে, button আবার click করুন (deactivate)

### সমস্যা ৫: "World AI Chat Auto not available"

**সমাধান**:

## 💡 Pro Tips

- যদি error হয়
- Button click করুন (deactivate)

- অথবা অন্য AI select করুন
