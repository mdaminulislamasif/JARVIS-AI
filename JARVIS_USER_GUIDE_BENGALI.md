## ⚙️ Settings & Configuration (সেটিংস)

### API Keys Setup:
1. **Gemini API Key** (AI responses এর জন্য):
   - Copy your Gemini API key
   - Paste in the "API Key" field
   - Click "SYNC KEY" button
   - অথবা clipboard এ copy করলে auto-apply হবে

2. **OpenAI API Key** (Optional):
   - For GPT models
   - Save in `Desktop/ai/openai_config.txt`

### Voice Settings:
- **Bengali Voice**: "Bangla Voice" checkbox চালু করুন
- **Agent Mode**: "Agent Mode" checkbox চালু করুন
- **Streaming Mode**: "Streaming" checkbox চালু করুন

### 3D Face Settings:
- **Enable 3D Face**: "3D FACE" button click করুন
- Face window আলাদা খুলবে

## 🐛 Troubleshooting (সমস্যা সমাধান)

### Problem 1: Jarvis শুরু হচ্ছে না
**Solution:**
```bash
# Check Python installed:
python --version

# Install dependencies:
pip install -r requirements.txt

# Run Jarvis:
python jarvis_panel.py
```

### Problem 2: Voice command কাজ করছে না
**Solution:**
- Microphone permission check করুন
- "LISTEN" button click করুন
- Microphone test করুন: Settings → Sound → Input

### Problem 3: AI response আসছে না
**Solution:**
- API key check করুন
- Internet connection check করুন
- API key valid কিনা verify করুন
- Console এ error message দেখুন

### Problem 4: Commands execute হচ্ছে না
**Solution:**
- Command spelling check করুন
- Bengali/English সঠিক ভাষায় লিখছেন কিনা check করুন
- Console এ error log দেখুন
