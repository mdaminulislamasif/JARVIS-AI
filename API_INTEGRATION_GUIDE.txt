  🚀 HOW TO ADD API KEYS
════════════════════════════════════════════════════════════════════════════

STEP 1: Choose APIs You Want
─────────────────────────────────────────────────────────────────────────
  - Start with FREE tier APIs
  - Recommended for beginners:
    ✅ Google Gemini (free tier)
    ✅ OpenWeatherMap (free tier)
    ✅ Remove.bg (free tier)
    ✅ SendGrid (free tier)

STEP 2: Get API Keys
─────────────────────────────────────────────────────────────────────────
  1. Visit the API website (links above)
  2. Sign up for free account
  3. Go to API settings/dashboard
  4. Generate API key
  5. Copy the API key (keep it secret!)

STEP 3: Configure in JARVIS
─────────────────────────────────────────────────────────────────────────
  1. Launch JARVIS
  2. Go to Settings → API Configuration
  3. Paste your API keys
  4. Save settings
  5. Done! JARVIS will automatically use APIs

STEP 4: Test API Features
─────────────────────────────────────────────────────────────────────────
  Try commands like:
  - "Generate an image of a sunset" (needs OpenAI/Stability AI)
  - "What's the weather today?" (needs OpenWeatherMap)
  - "Translate this to Spanish" (needs Google Translate/DeepL)
  - "Remove background from this image" (needs Remove.bg)

   - See how much you're spending
