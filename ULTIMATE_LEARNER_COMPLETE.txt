TECHNICAL DETAILS / প্রযুক্তিগত বিবরণ:
---------------------------------------

**Chrome Integration:**
- Finds Chrome on system
- Uses os.startfile() for Windows
- Opens Chrome with search URL
- Falls back to default browser if Chrome not found

**Learning Sources:**
- Wikipedia API (primary)
- Google Search (web scraping)
- Stack Overflow (programming topics)
- Web articles (general topics)

**Database:**
- ultimate_knowledge table
  - topic, category, content, source_url
  - word_count, importance, tags, learned_date
- learning_sessions table
  - topics_learned, total_words, duration_seconds

**Dependencies:**
- requests - HTTP requests
- beautifulsoup4 - Web scraping
- sqlite3 - Database (built-in)
