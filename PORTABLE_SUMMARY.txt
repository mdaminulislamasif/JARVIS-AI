TROUBLESHOOTING / সমস্যা সমাধান
================================================================================

PROBLEM 1: JARVIS won't start / JARVIS চালু হচ্ছে না
SOLUTION / সমাধান:
  1. Check all files are present / সব ফাইল আছে কিনা চেক করুন
  2. Run as Administrator / অ্যাডমিনিস্ট্রেটর হিসেবে চালান
  3. Check antivirus / অ্যান্টিভাইরাস চেক করুন

PROBLEM 2: Database error / ডাটাবেস ত্রুটি
SOLUTION / সমাধান:
  1. Delete jarvis_memory.db
  2. Restart JARVIS.exe
  3. New database will be created / নতুন ডাটাবেস তৈরি হবে

PROBLEM 3: Missing modules / মডিউল নেই
SOLUTION / সমাধান:
  1. Ensure core/ and engine/ folders present / core/ এবং engine/ ফোল্ডার আছে নিশ্চিত করুন
  2. Re-copy entire folder / সম্পূর্ণ ফোল্ডার পুনরায় কপি করুন

PROBLEM 4: Antivirus blocking / অ্যান্টিভাইরাস ব্লক করছে
SOLUTION / সমাধান:
  1. Add JARVIS.exe to exclusions / JARVIS.exe এক্সক্লুশনে যোগ করুন
  2. Temporarily disable antivirus / সাময়িকভাবে অ্যান্টিভাইরাস বন্ধ করুন
