## 🚀 Installation

### Prerequisites

- **Python 3.8+** ([Download](https://www.python.org/downloads/))
- **Git** ([Download](https://git-scm.com/downloads))
- **Modern Web Browser** (Chrome, Firefox, Edge)

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/jarvis-ai.git
cd jarvis-ai
```

### Install Dependencies

```bash
# Install Python packages
pip install -r requirements.txt

# Install voice engine (optional but recommended)
pip install pyttsx3

# Install Flask for voice server (optional)
pip install flask flask-cors
```

### Windows Quick Install

```batch
# Double-click to install
SETUP_JARVIS_COMPLETE.bat
```

## 🎯 Quick Start

### Method 1: Python GUI (Recommended)

```bash
# Run the main Python GUI
python jarvis_antigravity_panel_python.py
```

Or double-click:
```
RUN_PYTHON_PANEL.bat
```

### Method 2: HTML Panel

```bash
# Open in browser
start antigravity_panel.html
```

Or double-click:
```
OPEN_ANTIGRAVITY_PANEL.bat
```

### Method 3: Voice Control

```bash
# Start voice engine
python jarvis_voice_engine.py
```

Or double-click:
```
RUN_JARVIS_VOICE.bat
```

### Method 4: 3D Face Viewer

```bash
# Open colored 3D face
start JARVIS_3D_FACE_COLORED.html
```

Or double-click:
```
OPEN_COLORED_JARVIS_FACE.bat
```
