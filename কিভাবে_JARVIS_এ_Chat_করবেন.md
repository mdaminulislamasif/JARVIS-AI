## 🚀 ধাপে ধাপে Chat করার পদ্ধতি

### ধাপ ১: JARVIS Panel খুলুন ✅
```bash
START_JARVIS.bat
```
**Status**: ✅ আপনি এটা করেছেন! Panel খোলা আছে।

---

### ধাপ ২: Scroll করুন নিচে ⬇️

**এটাই মূল সমস্যা!** Chat box **একদম নিচে** থাকে।

**কিভাবে scroll করবেন**:

#### পদ্ধতি A: Mouse Wheel
```
1. Mouse pointer JARVIS panel এ রাখুন
2. Mouse wheel scroll করুন নিচে
3. একদম নিচে যান
4. Chat box দেখা যাবে!
```

#### পদ্ধতি B: Scrollbar
```
1. Panel এর ডান পাশে scrollbar দেখুন
2. Scrollbar টানুন নিচে
3. একদম নিচে নিয়ে যান
4. Chat box দেখা যাবে!
```

#### পদ্ধতি C: Keyboard
```
1. Panel এ click করুন
2. End key চাপুন (একদম নিচে যাবে)
3. অথবা Page Down চাপুন কয়েকবার
4. Chat box দেখা যাবে!
```

---

### ধাপ ৩: Chat Box এ Click করুন 🖱️

Chat box দেখতে এরকম:
```
┌─────────────────────────────────────────────────┐
│ Type your message here...                       │
│ ▌                                               │
└─────────────────────────────────────────────────┘
```

**করুন**:
1. Chat box এর ভিতরে click করুন
2. Cursor (▌) দেখা যাবে
3. এখন type করতে পারবেন!

## 🔧 Troubleshooting

### সমস্যা ১: Chat box দেখা যাচ্ছে না

**কারণ**: Scroll করেননি নিচে

**সমাধান**:
```
1. Mouse wheel scroll করুন নিচে
2. অথবা End key চাপুন
3. অথবা scrollbar টানুন নিচে
4. একদম নিচে যান!
```

---

### সমস্যা ২: Window ছোট, সব দেখা যাচ্ছে না

**সমাধান**:
```
1. Window maximize করুন
   - Maximize button click করুন
   - অথবা double-click title bar এ

2. অথবা window resize করুন বড় করে
   - Window এর corner টানুন
```

---

## 💡 Pro Tips

### Tip 1: Keyboard Shortcut
```
End key = একদম নিচে যাবে (chat box এ)
Home key = একদম উপরে যাবে
Page Down = নিচে scroll
Page Up = উপরে scroll
```

### Tip 2: Quick Focus
```
Tab key চাপুন = Chat box এ focus যাবে
```

### Tip 3: Multi-line Message
```
Shift + Enter = New line (message send হবে না)
Enter = Message send হবে
```

### Tip 4: Clear Output
```
Ctrl + L = Output area clear করবে
```
