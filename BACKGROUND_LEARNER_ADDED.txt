================================================================================
  🚀 HOW TO USE - কিভাবে ব্যবহার করবেন
================================================================================

STEP 1: Install Dependencies (if needed)
   
   pip install -r requirements_background.txt
   
   Or just run Start_JARVIS_Background.bat (auto-installs)


STEP 2: Start JARVIS Background Learner
   
   METHOD A: Double-click Start_JARVIS_Background.bat
   METHOD B: python jarvis_background_learner.py


STEP 3: Work Normally
   
   • Code in VS Code, PyCharm, etc.
   • Browse GitHub, YouTube, StackOverflow
   • Run commands in terminal
   • Copy code snippets
   
   JARVIS will learn automatically!


STEP 4: Check What JARVIS Learned
   
   Press Ctrl+L to see learned activities


STEP 5: View Statistics
   
   Press Ctrl+S to see app usage stats


STEP 6: Stop When Done
   
   Press Ctrl+Q to stop JARVIS

   • All data stored locally in SQLite database
   • No cloud upload
   • No internet connection required
   • Only you have access

✅ CONTROL:
   • Stop anytime with Ctrl+Q
   • Delete learned data anytime
   • Full transparency
   • You own your data

================================================================================
  🐛 TROUBLESHOOTING - সমস্যা সমাধান
================================================================================

ISSUE: "Missing dependency" error
FIX: pip install -r requirements_background.txt

ISSUE: Hotkeys not working
FIX: Run as Administrator (Right-click → Run as administrator)

ISSUE: Not learning anything
FIX: Make sure you're actively using apps. JARVIS learns from activity.

ISSUE: Want to stop
FIX: Press Ctrl+Q or close terminal window

ISSUE: Database locked
FIX: Close other JARVIS components, then restart

ISSUE: High CPU usage
FIX: Increase monitoring interval in code (change time.sleep(5) to higher value)
