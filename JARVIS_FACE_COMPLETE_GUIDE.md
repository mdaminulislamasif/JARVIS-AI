## ❓ সাধারণ প্রশ্ন (FAQ)

### Q1: কোন পদ্ধতি সবচেয়ে ভালো?
**A:** Browser version সবচেয়ে সহজ এবং দ্রুত। Professional quality এর জন্য Blender version।

### Q2: Blender install করতে হবে কি?
**A:** Browser version এ না। Blender version এ হ্যাঁ।

### Q3: কতক্ষণ সময় লাগবে?
**A:** 
- Browser version: 1-2 minutes
- Blender automatic: 2-3 minutes
- Blender with install: 10-15 minutes

### Q4: Export করা file কোথায় পাব?
**A:** 
- Browser version: Downloads folder
- Blender version: Current directory + JARVIS folders

### Q5: Lip sync কিভাবে test করব?
**A:** Browser version এ shape key buttons click করুন। Blender version এ Shape Keys panel ব্যবহার করুন।

### Q6: GLB file কিভাবে use করব?
**A:** আপনার JARVIS application এ import করুন। Three.js, Babylon.js, Unity, Unreal Engine সব support করে।

### Q7: Browser version এ কোন browser ব্যবহার করব?
**A:** Chrome, Firefox, Edge - যেকোনো modern browser।

### Q8: Blender version এ error হলে কি করব?
**A:** Browser version ব্যবহার করুন অথবা `INSTALL_BLENDER_AND_CREATE_FACE.bat` চালান।
