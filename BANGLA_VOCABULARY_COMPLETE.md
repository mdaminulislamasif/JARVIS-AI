## 🔧 কিভাবে কাজ করে:

### Database Structure:
**jarvis_bangla_vocabulary.db** contains:

**vocabulary table**:
- id (PRIMARY KEY)
- bangla_word (UNIQUE)
- english_meaning
- category
- example_bangla
- example_english
- added_at (TIMESTAMP)

### Search Algorithm:
1. User provides search term
2. Search in bangla_word column
3. Search in english_meaning column
4. Return matching results
5. Show with category

### Category Organization:
- 14 main categories
- Words organized by meaning
- Easy to browse
- Quick access
