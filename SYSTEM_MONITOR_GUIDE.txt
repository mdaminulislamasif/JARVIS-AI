এই feature দিয়ে আপনি দেখতে পারবেন JARVIS আপনার system এ কি করছে!
This feature lets you see what JARVIS is doing on your system!


🚀 কিভাবে ব্যবহার করবেন / HOW TO USE
---------------------------------------

METHOD 1: JARVIS Panel থেকে (সবচেয়ে সহজ)
------------------------------------------

1. JARVIS চালু করুন (JARVIS.bat)
   Start JARVIS (JARVIS.bat)

2. Left sidebar এ "🔍 SYSTEM MONITOR" button এ click করুন
   Click the "🔍 SYSTEM MONITOR" button in left sidebar

3. অথবা terminal এ টাইপ করুন: monitor
   Or type in terminal: monitor

4. আপনি দেখবেন:
   You will see:
   - CPU, Memory, Disk usage
   - JARVIS status
   - Running processes
   - Recent activities


METHOD 2: Standalone Script (আলাদা window এ)
---------------------------------------------

Option A: Single Snapshot (একবার দেখুন)
   
   Double-click: JARVIS_MONITOR.bat
   Then choose option 1

   অথবা / Or:
   python jarvis_system_monitor.py


Option B: Live Monitor (continuously updates)
   
   Double-click: JARVIS_MONITOR.bat
   Then choose option 2

   অথবা / Or:
   python jarvis_system_monitor.py live

   (Press Ctrl+C to stop / বন্ধ করতে Ctrl+C চাপুন)


📊 কি কি দেখাবে / WHAT IT SHOWS
---------------------------------

✅ System Resources / সিস্টেম রিসোর্স:
   - CPU usage (কত % CPU ব্যবহার হচ্ছে)
   - Memory usage (কত RAM ব্যবহার হচ্ছে)
   - Disk usage (কত disk space ব্যবহার হচ্ছে)

✅ JARVIS Status / JARVIS অবস্থা:
   - Brain status (Brain কাজ করছে কিনা)
   - Voice engine status (Voice engine ready কিনা)
   - Network status (Internet connected কিনা)
   - Files status (সব important files আছে কিনা)

✅ Recent Activities / সাম্প্রতিক কার্যক্রম:
   - JARVIS কি কি করেছে (last 5 activities)

✅ Running Processes / চলমান প্রসেস:
   - কোন কোন JARVIS processes চলছে
   - তারা কত CPU/Memory ব্যবহার করছে


💡 EXAMPLES / উদাহরণ
---------------------

Example 1: Check system status from JARVIS
JARVIS থেকে system status দেখুন:

   1. JARVIS চালু করুন
   2. টাইপ করুন: monitor
   3. দেখুন full system report


Example 2: Live monitoring while JARVIS works
JARVIS কাজ করার সময় live monitoring:

   1. JARVIS চালু করুন (একটা window এ)
   2. JARVIS_MONITOR.bat চালু করুন (আরেকটা window এ)
   3. Option 2 select করুন (Live Monitor)
   4. এখন JARVIS এ commands দিন
   5. Monitor window এ দেখুন কি হচ্ছে!


Example 3: Quick system check
দ্রুত system check:

   Double-click: JARVIS_MONITOR.bat
   Choose option 1
   দেখুন current status

   When JARVIS feels slow

   When you want to see resource usage

   When you want to check if everything is OK

   When you want to debug

   When you're curious what JARVIS is doing

🔧 TROUBLESHOOTING / সমস্যা সমাধান
-----------------------------------

Problem: Monitor button না দেখা যাচ্ছে
Solution: JARVIS restart করুন

Problem: "Module not found" error
Solution: python jarvis_system_monitor.py চালান

Problem: Live monitor কাজ করছে না
Solution: Ctrl+C চেপে বন্ধ করুন, আবার চালান
