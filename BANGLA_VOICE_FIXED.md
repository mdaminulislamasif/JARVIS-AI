### 2. Volume (শব্দের মাত্রা)

## 🚀 কিভাবে ব্যবহার করবেন (How to Use)

### Step 1: JARVIS চালু করুন
```bash
python jarvis_panel.py
```

### Step 2: Bengali Voice Selector খুলুন
1. সাইডবারে scroll করুন
2. "[ VOICE CONTROL ]" section খুঁজুন
3. "🇧🇩 BENGALI VOICE SELECTOR" বাটনে ক্লিক করুন

### Step 3: Voice বেছে নিন
1. ৪টি voice থেকে একটি select করুন:
   - 👩 Nabanita (Female) - সুপারিশকৃত!
   - 👨 Pradeep (Male) - সুপারিশকৃত!
   - 👨 Bashkar (Male)
   - 👩 Tanishaa (Female)

### Step 4: Test করুন
1. Test text box এ বাংলা লিখুন
2. "🔊 TEST SELECTED VOICE" বাটনে ক্লিক করুন
3. Voice শুনুন
4. পছন্দ না হলে অন্য voice try করুন

### Step 5: Save করুন
1. পছন্দের voice select করুন
2. "✅ SAVE & APPLY" বাটনে ক্লিক করুন
3. JARVIS restart করুন
