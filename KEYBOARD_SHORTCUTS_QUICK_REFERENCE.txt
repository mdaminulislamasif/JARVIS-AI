Problem: Shortcuts not working
সমস্যা: Shortcuts কাজ করছে না

Solution:
সমাধান:
1. Check if JARVIS window has focus
   দেখুন JARVIS window focus এ আছে কিনা

2. Look for "✅ Keyboard Shortcuts activated!" message
   "✅ Keyboard Shortcuts activated!" message খুঁজুন

3. Restart JARVIS
   JARVIS restart করুন

4. Check if another program is using the same shortcut
   দেখুন অন্য program একই shortcut ব্যবহার করছে কিনা
