═══════════════════════════════════════════════════════════════════════════════
💡 WHAT IS is_admin? / is_admin কি?
═══════════════════════════════════════════════════════════════════════════════

PURPOSE:
───────────────────────────────────────────────────────────────────────────────
Checks if JARVIS is running with Administrator privileges on Windows

WHY IT'S NEEDED:
───────────────────────────────────────────────────────────────────────────────
- Some operations require admin rights
- System-level operations need elevated privileges
- Better error handling when admin rights are needed

HOW IT WORKS:
───────────────────────────────────────────────────────────────────────────────
1. Uses Windows API (ctypes.windll.shell32.IsUserAnAdmin())
2. Returns True if running as admin
3. Returns False if running as normal user
4. Returns False if check fails (safety fallback)
