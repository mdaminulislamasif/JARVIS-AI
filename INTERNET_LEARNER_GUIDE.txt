NEW FEATURE / নতুন ফিচার:
--------------------------
✅ JARVIS can now learn from the internet automatically!
✅ JARVIS এখন স্বয়ংক্রিয়ভাবে internet থেকে শিখতে পারে!

WHAT IT DOES / এটা কি করে:
---------------------------

1. **Searches Internet** / Internet এ খোঁজে
   - Wikipedia থেকে তথ্য খোঁজে
   - Web search করে
   - সেরা তথ্য খুঁজে বের করে

2. **Learns Content** / বিষয়বস্তু শেখে
   - তথ্য পড়ে এবং বুঝে
   - গুরুত্বপূর্ণ অংশ extract করে
   - সংক্ষিপ্ত করে

3. **Saves to Memory** / মেমরিতে সংরক্ষণ করে
   - JARVIS knowledge base এ save করে
   - পরে ব্যবহারের জন্য রাখে
   - সহজে খুঁজে পাওয়ার জন্য organize করে

4. **Uses Knowledge** / জ্ঞান ব্যবহার করে
   - প্রশ্নের উত্তর দেয়
   - তথ্য প্রদান করে
   - সাহায্য করে

HOW TO USE / কিভাবে ব্যবহার করবেন:
------------------------------------

METHOD 1: Command Line / কমান্ড লাইন
--------------------------------------

1. **Learn a topic** / একটি বিষয় শিখুন:
   ```
   python jarvis_internet_learner.py learn Python
   python jarvis_internet_learner.py learn "Artificial Intelligence"
   python jarvis_internet_learner.py learn "Machine Learning"
   ```

2. **List learned topics** / শেখা বিষয়ের তালিকা:
   ```
   python jarvis_internet_learner.py list
   ```

3. **Search learned knowledge** / শেখা জ্ঞান খুঁজুন:
   ```
   python jarvis_internet_learner.py search Python
   python jarvis_internet_learner.py search AI
   ```

4. **Show statistics** / পরিসংখ্যান দেখান:
   ```
   python jarvis_internet_learner.py stats
   ```

METHOD 2: Interactive Mode / ইন্টারঅ্যাক্টিভ মোড
-------------------------------------------------

Run without arguments / arguments ছাড়া চালান:
```
python jarvis_internet_learner.py
```

Then use commands / তারপর commands ব্যবহার করুন:
- `learn Python` - Learn about Python
- `list` - List learned topics
- `search AI` - Search learned knowledge
- `stats` - Show statistics
- `exit` - Exit

METHOD 3: Through JARVIS Offline Brain / JARVIS Offline Brain এর মাধ্যমে
--------------------------------------------------------------------------

1. **Learn from internet** / Internet থেকে শিখুন:
   ```
   learn from internet Python
   internet learn Artificial Intelligence
   internet sikbo Machine Learning
   internet theke sikbo Data Science
   ```

2. **List learned topics** / শেখা বিষয়ের তালিকা:
   ```
   learned topics
   learned list
   sikha list
   ```

3. **Search learned knowledge** / শেখা জ্ঞান খুঁজুন:
   ```
   search learned Python
   sikha search AI
   ```

4. **Learning statistics** / শেখার পরিসংখ্যান:
   ```
   learning stats
   sikha stats
   learning statistics
   ```

TROUBLESHOOTING / সমস্যা সমাধান:
---------------------------------

Problem 1: "Could not learn from internet"
Solution: Check internet connection

Problem 2: "Database error"
Solution: Make sure jarvis_memory.db exists

Problem 3: "No results found"
Solution: Learn more topics first

Problem 4: "Import error"
Solution: Install required packages:
```
pip install requests beautifulsoup4
```
