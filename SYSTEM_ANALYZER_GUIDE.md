## 💡 Tips & Tricks / টিপস এবং ট্রিকস

### 1. Ask Naturally
```
✅ Good: "How much RAM do I have?"
✅ Good: "আমার কত RAM আছে?"
✅ Good: "Tell me about my CPU"
✅ Good: "What's my disk space?"
```

### 2. Be Specific
```
✅ Specific: "How many files on my desktop?"
❌ Vague: "Files?"

✅ Specific: "What is my CPU usage?"
❌ Vague: "Usage?"
```

### 3. Use Keywords
```
Keywords that work:
- my, আমার (my)
- how much, কত (how much)
- how many, কতগুলো (how many)
- what is, কি (what is)
- tell me, বলো (tell me)
- show me, দেখাও (show me)
```

### 4. Mix Languages
```
✅ "আমার CPU কি?"
✅ "What is my RAM?"
✅ "আমার disk space কত?"
```
