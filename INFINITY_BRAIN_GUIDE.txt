🚀 HOW TO USE | কিভাবে ব্যবহার করবেন
================================================================================

METHOD 1: One-Click Launcher (Easiest)
পদ্ধতি ১: ওয়ান-ক্লিক লঞ্চার (সবচেয়ে সহজ)
----------------------------------------

1. Double-click: START_INFINITY_BRAIN.bat
   ডাবল-ক্লিক করুন: START_INFINITY_BRAIN.bat

2. Choose mode:
   মোড নির্বাচন করুন:
   • 1 = Demo Mode (Learn 5 sample topics)
        ডেমো মোড (৫টি নমুনা বিষয় শিখুন)
   • 2 = Interactive Mode (Ask anything!)
        ইন্টারঅ্যাক্টিভ মোড (যেকোনো কিছু জিজ্ঞাসা করুন!)

3. Done! JARVIS will learn from internet
   সম্পন্ন! জার্ভিস ইন্টারনেট থেকে শিখবে


METHOD 2: Command Line
পদ্ধতি ২: কমান্ড লাইন
----------------------------------------

# Demo Mode (Learn 5 sample topics)
python jarvis_infinity_brain.py --demo

# Interactive Mode (Ask anything!)
python jarvis_infinity_brain.py --interactive

# Learn specific topic
python jarvis_infinity_brain.py --learn "Quantum Computing"
python jarvis_infinity_brain.py --learn "Python Programming"
python jarvis_infinity_brain.py --learn "Bangladesh History"


METHOD 3: Interactive Mode Examples
পদ্ধতি ৩: ইন্টারঅ্যাক্টিভ মোড উদাহরণ
----------------------------------------

Start interactive mode:
ইন্টারঅ্যাক্টিভ মোড শুরু করুন:

python jarvis_infinity_brain.py --interactive

Then ask anything:
তারপর যেকোনো কিছু জিজ্ঞাসা করুন:

You: What is Artificial Intelligence?
You: Tell me about Quantum Computing
You: What is the capital of Bangladesh?
You: Explain Machine Learning
You: Who is Albert Einstein?

Type 'exit' to quit
'exit' টাইপ করুন বের হতে
