জার্ভিস ব্যাকগ্রাউন্ডে চলবে এবং আপনার কাজ দেখে শিখবে!

✅ আপনি কোন অ্যাপ ব্যবহার করছেন
✅ আপনি কি কোড লিখছেন
✅ আপনি কোথায় ব্রাউজ করছেন
✅ আপনি কি কমান্ড চালাচ্ছেন
✅ সব কিছু স্বয়ংক্রিয়ভাবে শিখে এবং সেভ করে!


🚀 কিভাবে চালাবেন:
================================================================================

পদ্ধতি ১: ডাবল ক্লিক (সবচেয়ে সহজ)
   
   📂 খুঁজুন: Start_JARVIS_Background.bat
   🖱️ ডাবল ক্লিক করুন
   ✅ জার্ভিস ব্যাকগ্রাউন্ডে চলবে!


পদ্ধতি ২: কমান্ড লাইন
   
   python jarvis_background_learner.py


পদ্ধতি ৩: Windows এর সাথে স্বয়ংক্রিয় শুরু
   
   1. Win+R চাপুন
   2. টাইপ করুন: shell:startup
   3. Start_JARVIS_Background.bat কপি করুন সেখানে
   4. Windows চালু হলেই জার্ভিস চালু হবে!


⌨️ হটকি (Hotkeys):
================================================================================

জার্ভিস চলার সময়:

🔹 Ctrl+Q : জার্ভিস বন্ধ করুন
🔹 Ctrl+S : পরিসংখ্যান দেখুন
🔹 Ctrl+L : জার্ভিস কি শিখেছে দেখুন


📚 জার্ভিস কি শিখে:
================================================================================

1️⃣ প্রোগ্রামিং
   • "User is coding in .py file"
   • "User is coding in .js file"
   • আপনি কোন ভাষায় কোড লিখছেন

2️⃣ ওয়েব ব্রাউজিং
   • "User is browsing: GitHub"
   • "User is browsing: YouTube"
   • আপনার আগ্রহের বিষয়

3️⃣ কমান্ড
   • "User used command: python test.py"
   • "User used command: git commit"
   • আপনার ব্যবহৃত কমান্ড

4️⃣ কোড স্নিপেট
   • কপি করা কোড
   • ফাংশন এবং ক্লাস

5️⃣ কাজের প্যাটার্ন
   • কোন অ্যাপ বেশি ব্যবহার করেন
   • কতক্ষণ কোড লিখেন
   • কতক্ষণ ব্রাউজ করেন
