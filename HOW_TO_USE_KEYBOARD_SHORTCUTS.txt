┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 1: START JARVIS                                                         │
│ ধাপ ১: JARVIS শুরু করুন                                                     │
└──────────────────────────────────────────────────────────────────────────────┘

    Open your terminal/command prompt and run:
    আপনার terminal/command prompt খুলুন এবং চালান:

    python jarvis_panel.py

    You will see this message:
    আপনি এই বার্তা দেখবেন:

    ✅ Keyboard Shortcuts activated!
    💡 Press Ctrl+H for shortcuts help

    This means shortcuts are ready to use!
    এর মানে shortcuts ব্যবহারের জন্য প্রস্তুত!

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 2: PRESS CTRL+H TO SEE ALL SHORTCUTS                                   │
│ ধাপ ২: সব শর্টকাট দেখতে Ctrl+H চাপুন                                        │
└──────────────────────────────────────────────────────────────────────────────┘

    Hold down CTRL key and press H:
    CTRL key ধরে রাখুন এবং H চাপুন:

    ┌──────┐     ┌────┐
    │ CTRL │  +  │ H  │
    └──────┘     └────┘

    This will show you the complete list of all 46 shortcuts!
    এটি আপনাকে সব 46 shortcuts এর সম্পূর্ণ তালিকা দেখাবে!

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 3: TRY SOME SHORTCUTS                                                   │
│ ধাপ ৩: কিছু শর্টকাট চেষ্টা করুন                                             │
└──────────────────────────────────────────────────────────────────────────────┘

    Try these easy shortcuts:
    এই সহজ shortcuts চেষ্টা করুন:

    1. Press F2
       F2 চাপুন
       → Takes a screenshot
       → একটি স্ক্রিনশট নেয়

    2. Press F5
       F5 চাপুন
       → Cleans your system
       → আপনার সিস্টেম পরিষ্কার করে

    3. Press F12
       F12 চাপুন
       → Opens system monitor
       → সিস্টেম মনিটর খোলে

    4. Press Ctrl+N
       Ctrl+N চাপুন
       → Scans your network
       → আপনার নেটওয়ার্ক স্ক্যান করে

    That's it! You're now using keyboard shortcuts!
    এটাই! আপনি এখন keyboard shortcuts ব্যবহার করছেন!

    PROBLEM: Shortcuts not working
    সমস্যা: Shortcuts কাজ করছে না

    SOLUTION:
    সমাধান:

    1. Make sure JARVIS window is active (click on it)
       নিশ্চিত করুন JARVIS window active আছে (এটিতে ক্লিক করুন)

    2. Look for "✅ Keyboard Shortcuts activated!" message
       "✅ Keyboard Shortcuts activated!" বার্তা খুঁজুন

    3. Try pressing Ctrl+H to test
       পরীক্ষা করতে Ctrl+H চাপার চেষ্টা করুন

    4. Restart JARVIS if needed
       প্রয়োজনে JARVIS restart করুন
