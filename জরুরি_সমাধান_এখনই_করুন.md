## 💡 Alternative: Command Line Mode

যদি panel freeze হতেই থাকে, **panel ছাড়া** JARVIS ব্যবহার করুন:

### তৈরি করুন: `jarvis_cli.py`

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""JARVIS Command Line Interface"""

print("=" * 70)
print("JARVIS CLI - Command Line Interface")
print("=" * 70)
print()
print("Type 'exit' to quit")
print()

try:
    from jarvis_offline_brain import OfflineBrain
    brain = OfflineBrain()
    print("[OK] JARVIS Brain loaded")
except Exception as e:
    print(f"[X] Error loading brain: {e}")
    print("[!] Using simple responses")
    brain = None

print()

while True:
    try:
        query = input("You: ")
        
        if query.lower() in ['exit', 'quit', 'bye', 'q']:
            print("JARVIS: Goodbye!")
            break
        
        if not query.strip():
            continue
        
        if brain:
            response = brain.get_response(query)
        else:
            response = f"I heard: {query}"
        
        print(f"JARVIS: {response}")
        print()
        
    except KeyboardInterrupt:
        print("\nJARVIS: Goodbye!")
        break
    except Exception as e:
        print(f"[X] Error: {e}")
```

### চালান:
```bash
python jarvis_cli.py
```

### ব্যবহার করুন:
```
You: Hello JARVIS
JARVIS: Hello! How can I help you?

You: What is Python?
JARVIS: Python is a programming language...

You: exit
JARVIS: Goodbye!
```

**এটি panel ছাড়া কাজ করবে!** ✅
