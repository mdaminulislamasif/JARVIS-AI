🎤 VOICE TIPS:
--------------

1. SPEAK CLEARLY
   - Speak at normal pace
   - Don't speak too fast or too slow
   - Use clear pronunciation

2. QUIET ENVIRONMENT
   - Reduce background noise
   - Close windows if outside is noisy
   - Turn off loud music

3. MICROPHONE POSITION
   - Keep microphone close (but not too close)
   - Speak directly into microphone
   - Avoid covering microphone

4. WAIT FOR "LISTENING"
   - Always wait for 🎤 Listening... message
   - Don't speak before it's ready
   - Give it a moment to process

5. NATURAL LANGUAGE
   - Speak naturally
   - Use complete sentences
   - Be specific about what you want


🔧 TROUBLESHOOTING:
-------------------

Problem: "Microphone not detected"
Solution: 
  1. Check if microphone is plugged in
  2. Check Windows microphone settings
  3. Test microphone in Windows settings
  4. Try different USB port

Problem: "Could not understand audio"
Solution:
  1. Speak more clearly
  2. Reduce background noise
  3. Move closer to microphone
  4. Speak louder

Problem: "Speech recognition error"
Solution:
  1. Check internet connection (Google Speech API needs internet)
  2. Try again in a few seconds
  3. Restart the program

Problem: "PyAudio installation failed"
Solution:
  1. Run: pip install pipwin
  2. Run: pipwin install pyaudio
  OR
  3. Download PyAudio wheel from:
     https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
  4. Install with: pip install [downloaded-file].whl

Problem: "No speech detected"
Solution:
  1. Speak within 5 seconds of "Listening..."
  2. Check microphone volume in Windows
  3. Test microphone with option 2 in menu
