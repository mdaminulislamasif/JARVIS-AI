## 💡 Usage Examples / ব্যবহারের উদাহরণ

### Example 1: Recognize Files
```python
from jarvis_super_intelligence import SuperIntelligence

si = SuperIntelligence()

# Recognize different files
files = ["document.pdf", "song.mp3", "video_4k.mp4", "code.py", "photo.jpg"]

for file in files:
    result = si.recognize_anything(file)
    print(f"{file} -> {result['category']} ({result.get('quality', 'N/A')})")
```

**Output:**
```
document.pdf -> document (N/A)
song.mp3 -> audio (N/A)
video_4k.mp4 -> video (4K)
code.py -> code (N/A)
photo.jpg -> image (N/A)
```

### Example 2: Auto-Navigate Website
```python
from jarvis_autonomous_system import AutonomousSystem

auto = AutonomousSystem()

# Define actions
actions = [
    {'type': 'wait', 'seconds': 2},
    {'type': 'fill', 'selector': 'input[name="q"]', 'value': 'Python'},
    {'type': 'click', 'selector': 'button[type="submit"]'},
    {'type': 'wait', 'seconds': 3},
    {'type': 'collect_data'},
    {'type': 'screenshot', 'filename': 'results.png'}
]

# Execute
result = auto.auto_navigate_website("https://www.google.com", actions)
print(f"Status: {result['status']}")
print(f"Actions completed: {result['actions_completed']}")

auto.close()
```

from jarvis_super_intelligence import SuperIntelligence
