### How to Use Advanced Cache:

```python
from jarvis_advanced_cache import SmartCache

# Create cache
cache = SmartCache(max_size=1000, default_ttl=3600)

# Cache function results
@cache.cache_function(ttl=300)
def expensive_function(x, y):
    return x + y

# Use function (will be cached)
result = expensive_function(5, 3)
```

