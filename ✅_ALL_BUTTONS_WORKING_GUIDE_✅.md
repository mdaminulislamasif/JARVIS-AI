## 🐛 TROUBLESHOOTING | সমস্যা সমাধান

### Problem 1: Button Clicks But No Response
**Reason**: Command is processing  
**Solution**: Wait a few seconds, check terminal

### Problem 2: Error Message Appears
**Reason**: Missing dependency or permission  
**Solution**: 
- Read error message
- Install missing module
- Run as administrator if needed

### Problem 3: Button Opens Dialog
**Reason**: Some buttons open dialogs (TRANSLATE, GENERATE, etc.)  
**Solution**: This is normal! Fill in the dialog and click generate/translate

### Problem 4: Slow Response
**Reason**: Some commands take time (learning, scanning)  
**Solution**: Be patient, check progress in terminal
