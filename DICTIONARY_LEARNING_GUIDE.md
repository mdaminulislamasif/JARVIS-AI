## 🔧 Technical Details:

### Tree Learning Process:
```
1. Open https://www.english-bangla.com/browse/bntobn
   ↓
2. Extract all words from page
   ↓
3. Find all links on page
   ↓
4. Open each link in new tab
   ↓
5. Extract words from each tab
   ↓
6. Find links in those tabs
   ↓
7. Continue recursively
   ↓
8. Store all words in database
```

### Database Structure:
```sql
vocabulary table:
- id (PRIMARY KEY)
- bangla_word (UNIQUE)
- english_meaning
- category
- example_bangla
- example_english
- added_at (TIMESTAMP)
```
