HOW IT WORKS / কিভাবে কাজ করে:
--------------------------------

STEP 1: You give a topic / আপনি একটি বিষয় দেন
   Example: "learn from internet Python"

STEP 2: JARVIS searches internet / JARVIS internet এ খোঁজে
   - Tries Wikipedia first
   - Falls back to web search
   - Extracts best information

STEP 3: JARVIS learns content / JARVIS বিষয়বস্তু শেখে
   - Reads and understands
   - Extracts key information
   - Summarizes content

STEP 4: JARVIS saves to memory / JARVIS মেমরিতে সংরক্ষণ করে
   - Saves to knowledge_base table
   - Saves to internet_learned table
   - Organizes by category

STEP 5: JARVIS can use it / JARVIS এটা ব্যবহার করতে পারে
   - Answers questions
   - Provides information
   - Helps you

TECHNICAL DETAILS / প্রযুক্তিগত বিবরণ:
---------------------------------------

**InternetLearner Class:**
- search_and_learn(topic) - Learn from internet
- get_learned_topics() - List learned topics
- search_learned_knowledge(query) - Search knowledge
- get_statistics() - Show statistics
- learn_multiple_topics(topics_list) - Batch learning

**Database Tables:**
- internet_learned - Stores learned content
  - id, topic, content, source_url, learned_date, word_count, category
- knowledge_base - Quick access storage
  - topic, content

**Learning Sources:**
- Wikipedia API (primary)
- Web scraping (fallback)
- Google search (fallback)

**Dependencies:**
- requests - HTTP requests
- beautifulsoup4 - Web scraping
- sqlite3 - Database (built-in)
