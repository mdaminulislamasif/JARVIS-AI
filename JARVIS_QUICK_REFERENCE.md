# Jarvis Quick Reference Card

## 🚀 Quick Start
```bash
python jarvis_panel.py
```

## 💬 Basic Usage

### Text Chat:
Type in chat box → Press Enter

### Voice Chat:
Click "LISTEN" button → Speak → Jarvis responds

---

## 🎯 Most Used Commands

### Bengali:
```
"হ্যালো জার্ভিস"              → Greeting
"গুগলে সার্চ করো [topic]"    → Google search
"ক্লিন করো"                  → Clean system
"সিস্টেম স্ট্যাটাস"           → System stats
"ছবি তৈরি করো [description]" → Generate image
"স্ক্রিনশট নাও"               → Take screenshot
"ধন্যবাদ"                     → Thank you
```

### English:
```
"Hello Jarvis"                → Greeting
"search Google for [topic]"   → Google search
"clean"                       → Clean system
"stats"                       → System stats
"generate image [description]"→ Generate image
"screenshot"                  → Take screenshot
"thank you"                   → Thank you
```

---

## 📋 Command Categories

### 🔍 Search & Info:
- `search Google for [topic]`
- `what's the latest [topic]`
- `tell me about [topic]`

### 💻 System:
- `clean` - Clean system
- `stats` - System statistics
- `shutdown` - Shutdown PC
- `restart` - Restart PC
- `lock` - Lock PC

### 📁 Files:
- `screenshot` - Take screenshot
- `open explorer` - File explorer
- `task manager` - Task manager

### 🌐 Network:
- `scan network` - Scan network
- `wifi scan` - Scan WiFi
- `my ip` - Show IP

### 🎨 AI Generate:
- `generate text [prompt]`
- `generate image [description]`
- `generate audio [text]`
- `generate video [description]`

### 🔐 Security:
- `port scan [IP]`
- `firewall`
- `virus scan`

---

## ⚙️ Setup

### API Key:
1. Copy Gemini API key
2. Paste in "API Key" field
3. Click "SYNC KEY"

### Voice:
- Bengali: Check "Bangla Voice"
- English: Uncheck "Bangla Voice"

---

## 🐛 Quick Fixes

### Jarvis won't start:
```bash
pip install -r requirements.txt
python jarvis_panel.py
```

### Voice not working:
- Check microphone permission
- Click "LISTEN" button

### No AI response:
- Check API key
- Check internet connection
