# Requirements Document - JARVIS Automatic Coding Learning

## Introduction - ভূমিকা

This document specifies requirements for **JARVIS Automatic Coding Learning** system. JARVIS will automatically learn ALL programming languages, coding skills, frameworks, libraries, and best practices from the internet without waiting for commands. JARVIS will become a master programmer in ALL languages automatically.

এই ডকুমেন্টে **JARVIS Automatic Coding Learning** সিস্টেমের requirements উল্লেখ করা হয়েছে। JARVIS স্বয়ংক্রিয়ভাবে সব programming languages, coding skills, frameworks, libraries এবং best practices ইন্টারনেট থেকে শিখবে কোন command এর জন্য অপেক্ষা না করে। JARVIS স্বয়ংক্রিয়ভাবে সব language এ master programmer হয়ে উঠবে।
