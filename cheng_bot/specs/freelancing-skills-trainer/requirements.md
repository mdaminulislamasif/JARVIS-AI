# Requirements Document - Freelancing & Professional Skills Trainer

## Introduction - ভূমিকা

This document specifies requirements for JARVIS Freelancing & Professional Skills Trainer - enabling JARVIS to teach freelancing, bug fixing, programming, and all professional skills needed to earn money independently. JARVIS will teach users how to find work, fix bugs, write code, manage projects, and become successful freelancers.

এই ডকুমেন্টে JARVIS Freelancing & Professional Skills Trainer এর requirements উল্লেখ করা হয়েছে - JARVIS freelancing, bug fixing, programming এবং স্বাধীনভাবে অর্থ উপার্জনের জন্য প্রয়োজনীয় সব professional skills শেখাবে। JARVIS users কে শেখাবে কিভাবে কাজ খুঁজতে হয়, bug fix করতে হয়, code লিখতে হয়, project manage করতে হয় এবং সফল freelancer হতে হয়।

## Glossary - শব্দকোষ

- **Freelancing_Trainer**: System that teaches freelancing skills
- **Bug_Fix_Teacher**: Teaches bug fixing and debugging skills
- **Programming_Tutor**: Teaches programming and coding
- **Project_Manager_Trainer**: Teaches project management
- **Client_Communication_Coach**: Teaches client communication
- **Portfolio_Builder**: Helps build professional portfolio
- **Job_Finder**: Helps find freelancing jobs
- **Skill_Assessor**: Assesses current skill level
- **Learning_Path_Creator**: Creates personalized learning paths
- **Income_Optimizer**: Optimizes earning potential
