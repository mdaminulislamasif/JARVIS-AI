## 🎯 কিভাবে ব্যবহার করবেন?

### সহজ পদ্ধতি:

1. **JARVIS Panel খুলুন**
   ```bash
   python jarvis_panel.py
   ```

2. **"🌍 WORLD AI CHAT" button এ click করুন**
   - Button টি modules section এ আছে (সবুজ রঙের)

3. **AI Select করুন**
   - Dialog খুলবে
   - যেকোনো AI select করুন:
     - 🔷 Google Gemini
     - 🤖 ChatGPT
     - 🧠 Claude AI
     - 💬 Microsoft Copilot
     - 🔮 Perplexity AI
     - এবং আরও ৫টি AI!

4. **প্রশ্ন লিখুন**
   - Dialog এ আপনার প্রশ্ন type করুন
   - OK click করুন

5. **Browser এ paste করুন**
   - Browser automatically খুলবে
   - আপনার প্রশ্ন clipboard এ আছে
   - Ctrl+V চাপুন paste করতে

6. **Response copy করুন**
   - AI response দেওয়ার পর
   - Response select করে Ctrl+C চাপুন

7. **JARVIS এ paste করুন**
   - JARVIS dialog এ ফিরে আসুন
   - Ctrl+V চাপুন
   - SUBMIT click করুন

8. **সম্পন্ন!** ✅
   - JARVIS response দেখাবে
   - Response speak করবে
