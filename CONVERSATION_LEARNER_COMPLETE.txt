CONVERSATION TYPES:
───────────────────────────────────────────────────────────────────────────────
1. GREETINGS / অভিবাদন

   - How are you, What's up
   - হ্যালো, তুমি কেমন আছো

   - What is..., How does..., Can you explain...
   - Tell me about..., Why is...
   - কি, কিভাবে, কেন

3. RESPONSES / উত্তর
   - Answering questions naturally
   - Explaining concepts
   - Giving suggestions

4. EMOTIONS / আবেগ
   - Expressing happiness, sadness
   - Showing empathy
   - Comforting, encouraging

5. DAILY TALK / দৈনন্দিন কথা
   - Weather, food, hobbies
   - Work, family, plans
   - আবহাওয়া, খাবার, পরিকল্পনা

6. TECHNICAL TALK / প্রযুক্তিগত কথা
   - Programming concepts
   - Technology discussions
   - AI, ML explanations

7. BENGALI CONVERSATION / বাংলা কথোপকথন
   - Bengali greetings
   - Bengali questions
   - Bengali responses

═══════════════════════════════════════════════════════════════════════════════
📊 HOW IT WORKS / কিভাবে কাজ করে
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Learn from Wikipedia
  - Gets article about topic
  - Extracts sentences
  - Creates Q&A pairs
  - Saves to database

STEP 2: Learn Patterns
  - Learns greeting patterns
  - Learns question patterns
  - Learns response patterns
  - Learns Bengali patterns

STEP 3: Store in Database
  - Question + Answer pairs
  - Pattern type (greeting, question, etc.)
  - Language (English/Bengali)
  - Usage count (tracks popularity)

STEP 4: Use When Needed
  - User asks question
  - JARVIS searches database
  - Finds matching conversation
  - Responds naturally!

═══════════════════════════════════════════════════════════════════════════════
💾 DATABASE STRUCTURE / ডেটাবেস গঠন
═══════════════════════════════════════════════════════════════════════════════

TABLE: conversations_learned
───────────────────────────────────────────────────────────────────────────────
- id: Unique ID
- question: User question/input
- answer: JARVIS response
- context: Topic/context
- pattern_type: greeting, question, response, etc.
- language: english or bengali
- learned_at: Timestamp
- usage_count: How many times used

TABLE: sentence_patterns
───────────────────────────────────────────────────────────────────────────────
- id: Unique ID
- pattern: Sentence pattern template
- example: Example sentence
- category: Pattern category
- learned_at: Timestamp

📊 STATISTICS / পরিসংখ্যান
═══════════════════════════════════════════════════════════════════════════════

To see conversation statistics:
কথোপকথন statistics দেখতে:

TOTAL CONVERSATIONS:
  SELECT COUNT(*) FROM conversations_learned

BY PATTERN TYPE:
  SELECT pattern_type, COUNT(*) as count
  FROM conversations_learned
  GROUP BY pattern_type

BY LANGUAGE:
  SELECT language, COUNT(*) as count
  FROM conversations_learned
  GROUP BY language

MOST USED:
  SELECT question, answer, usage_count
  FROM conversations_learned
  ORDER BY usage_count DESC
  LIMIT 10
