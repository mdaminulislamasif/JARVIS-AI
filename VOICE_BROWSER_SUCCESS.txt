🆘 TROUBLESHOOTING:
-------------------

Problem: "Microphone not detected"
Solution: Check if microphone is plugged in and enabled

Problem: "Could not understand audio"
Solution: Speak more clearly, reduce background noise

Problem: "PyAudio installation failed"
Solution: 
  pip install pipwin
  pipwin install pyaudio

Problem: "Speech recognition error"
Solution: Check internet connection (needs Google API)
