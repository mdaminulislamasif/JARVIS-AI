### What Happens in Background:

#### Thread 1: Internet Learner
```
Every 30 seconds:
- Picks next topic
- Learns from Wikipedia
- Saves to database
- Moves to next topic
```

#### Thread 2: Ultimate Learner
```
Every 60 seconds:
- Picks next topic
- Opens Chrome
- Searches Google
- Learns from multiple sources
- Saves to database
```

#### Thread 3: Auto Learner
```
Every 90 seconds:
- Picks next topic
- Opens Chrome
- Learns word by word
- Learns paragraph by paragraph
- Saves to file and database
```

#### Thread 4: Tree Learner
```
Every 120 seconds:
- Picks next topic
- Opens search results in tree
- Learns from each level
- Builds tree structure
```

#### Thread 5: Tree Auto Learner
```
Every 150 seconds:
- Picks next topic
- Opens tree structure
- Auto learns from each page
- Combines tree + auto learning
```

#### Thread 6: Infinite Learner
```
Every 180 seconds:
- Picks next topic
- Opens ALL links infinitely
- Never stops crawling
- Maximum depth learning
```

#### Thread 7: DevTools Learner
```
Every 120 seconds:
- Picks next topic
- Opens Chrome DevTools
- Advanced learning
- Saves to database
```
